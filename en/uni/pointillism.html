Pointillist program (March-june 2020)

Goal: Implementing a program that applies a "pointillist" effect to an image, in other words that will generate an image using only small points of colour, based on an article by Hiroki Tokura, Yuki Kuroda, Yasuaki Ito and Koji Nakano. In this project, each point is elliptical (as opposed to the publication, in which they are square), choosing an orientation and colour that best transcribes the original image, by getting as close as possible to the gaussian filter that simulates human vision. To imitate a pointillist aspect and their use of colour, the shade of each "stroke" is randomly changed, especially in flat tint areas. Precision is a parameter, that needs to be chosen proportionally to the size of the original image (results below respectively correspond to sizes 128*128, 256*256 and 512*512 for the same parameter value). The final algorithm has been implemented both in Python and C++.

Context: First year of master's in Artificial Intelligence, in Erasmus, at University of Malta.
